# ZipRide 2.0 🚗

ZipRide 2.0 is a frontend ride booking web application developed to provide a simple and user-friendly ride management experience.

## About the Project
This project focuses on building a responsive and interactive frontend interface with features like user authentication, dashboard, and ride booking UI.

## Features
- User Registration & Login
- User Dashboard
- Ride booking interface
- Responsive design
- Interactive UI components

## Technologies Used
- React.js
- JavaScript
- HTML5
- CSS3
- Vite
